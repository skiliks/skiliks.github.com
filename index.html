Skiliks PEAR Channel

Registering the channel: 
pear channel-discover skiliks.github.com
Listing available packages: 
pear remote-list -c skiliks
Installing a package: 
pear install skiliks/package_name
Installing a specific version/stability: 
pear install skiliks/package_name-1.0.0
pear install skiliks/package_name-beta
Receiving updates via a feed: 
http://skiliks.github.com/feed.xml
